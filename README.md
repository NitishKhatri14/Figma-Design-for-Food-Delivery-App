# Figma-Design-for-Food-Delivery-App
Food Point is an food delivery and restaurant booking mobile app UI/UX designed in Figma. The application provides smart food recommendations through AI Dietitian along with features like online food ordering, dine-in booking, food scheduling, and order tracking. The project focuses on creating a modern, user-friendly, attractive mobile interface.

## Features
- AI Dietitian Assistant
- Food Delivery System
- Restaurant Discovery
- Food Scheduling
- Dine-In Table Booking
- Cart Management
- Secure Online Payments
- Live Delivery Tracking
- Notifications
- Cake & Dessert Ordering
